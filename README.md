# Fechamento GR

This is a Flask-based project for managing "Fechamento GR".

## Quick setup

1. Create a virtual environment and activate it:

   Windows (PowerShell):
   ```powershell
   python -m venv .venv
   .\.venv\Scripts\Activate.ps1
   ```

2. Install dependencies (if you have a requirements.txt):
   ```powershell
   pip install -r requirements.txt
   ```

3. Make sure you do NOT commit your `.env` file (it's in `.gitignore`).

## To create the repo and push to GitHub
Follow these steps (I'll run them for you if you want):

- `git init`
- `git add .`
- `git commit -m "Initial commit"`
- Create a GitHub repo and run:
  `git remote add origin https://github.com/USERNAME/REPO.git`
  `git branch -M main`
  `git push -u origin main`
